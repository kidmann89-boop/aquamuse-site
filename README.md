# Aquamuse

Static website for Aquamuse, ready for GitHub Pages.

## Publish on GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open `Settings -> Pages`.
4. In `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait for GitHub to publish the site.

After publishing, replace `https://aquamuse.example` in `sitemap.xml` and `index.html` with the real GitHub Pages URL.
